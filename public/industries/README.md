# Industry hero photography

Placeholder slot for licensed industry photography used in the homepage
Service-Intensive tab strip. The five tabs and their image keys are defined in
`/data/industries.ts`.

Until licensed assets are dropped in here, the tab images are pulled from
Unsplash workspace/team photography (free licence) so the page is never
broken. Replace each Unsplash URL in `/data/industries.ts` with a local path
under this folder when licensed photography is ready:

- `service-intensive.jpg` — Service-Intensive Software
- `security.jpg`           — Security
- `it-services.jpg`        — IT Services
- `healthcare.jpg`         — Healthcare & Life Sciences
- `connected.jpg`          — Connected Businesses

Recommended specs:

- Format: JPG (or AVIF/WebP) — wide editorial photography
- Aspect: 16:9 (e.g. 1920×1080)
- Crop: leave the bottom-left third low-contrast (testimonial overlay sits there)
- Subject: real workspaces / teams (no stock-photo clichés)

**Replace with licensed photography before launch.**
